¿Por qué se calculan así las zonas?

Las zonas de entrenamiento se basan en el umbral de lactato (LTHR), que representa la intensidad máxima que puedes sostener de forma prolongada.

Cada zona corresponde a un rango fisiológico diferente:

  • Z1: Recuperación (bajo estrés)
  • Z2: Base aeróbica (uso eficiente de grasa)
  • Z3: Ritmo controlado
  • Z4: Umbral (alto rendimiento)
  • Z5: VO₂ máx (máxima intensidad)

¿Cómo usar estas zonas?

Utiliza Z2 para mejorar resistencia, Z4 para aumentar tu rendimiento y Z5 para intervalos intensos.
