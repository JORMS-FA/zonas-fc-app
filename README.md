# 🧠 Zonas FC PRO

Calculadora de zonas de frecuencia cardíaca basada en el umbral de lactato (LTHR).

## 🚀 Características
- Cálculo automático de zonas
- Basado en fisiología real
- Interfaz simple y rápida
- Funciona como app web (PWA)

## 📊 Cómo funciona
Las zonas se calculan a partir del umbral (LTHR):

- Z1: <75%
- Z2: 75–85%
- Z3: 85–92%
- Z4: 92–99%
- Z5: ≥100%

## 🎯 Objetivo
Resolver el problema de calcular zonas de entrenamiento de forma práctica y accesible.
